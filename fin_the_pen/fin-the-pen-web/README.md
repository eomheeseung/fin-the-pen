## ⚙️ Run Locally

  실행 전, Node.js 설치를 반드시 해주세요.
  
  vscode에서 실행하시면 편합니다.


Clone the project

```bash
  git clone https://github.com/soyoung125/fin-the-pen-web.git
```

Go to the project directory

```bash
  cd fin-the-pen-web
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
